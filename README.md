# ChatGBT Demo

This Android app shows a demo of a chat with ChatGPT.

# Technologies used in this demo:
* Kotlin
* Ktor
* Kotlinx-Serialization
* Delegates
* Typealias
* Coroutines
* Flow
* Hilt
* Coil
* View Binding
* MVVM
* Clean Architecture
